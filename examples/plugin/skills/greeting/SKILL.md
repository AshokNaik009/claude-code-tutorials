# Greeting Skill

Greet the user named "$ARGUMENTS" warmly and ask how you can help them today.

If no name is provided, give a general friendly greeting.

Make the greeting:
- Warm and encouraging
- Brief (2-3 sentences)
- End with a question about what they'd like to work on
